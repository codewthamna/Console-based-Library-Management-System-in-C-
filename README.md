# Library Management System (C++)

## 📌 About the Project
This project is a **console-based Library Management System** built using **C++ and Object-Oriented Programming (OOP)**.  
It allows users to manage a collection of books by performing operations like **adding, displaying, searching, updating, and deleting books**.  

The main goal is to practice **classes, arrays of objects, conditional logic, and menu-driven programming**.

---

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (Classes & Objects)
- Arrays
- Conditional Statements (if–else)
- Console Input/Output

---

## ✨ Features
- Add multiple books to the library  
- Display all available books  
- Search for a book by **Book ID**  
- Update book details (name, ID, availability)  
- Delete a book by **Book ID**  
- Handles invalid input gracefully  
- Menu-driven interface for easy navigation  

---

## ⚙️ How It Works
1. User runs the program and sees a menu of options.  
2. User chooses an action (Add, Display, Search, Update, Delete, Exit).  
3. The program performs the chosen action using **OOP classes** and **arrays of objects**.  
4. Book details (name, ID, availability) are stored in memory and displayed as needed.  
5. User can exit when finished.
